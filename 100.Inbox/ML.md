## 분류 (<span class="sub-word">Classification</span>)
여러 개의 종류 혹은 클래스(<span class="sub-word">class</span>) 중 하나를 구별해 내는 문제
2개의 클래스 중 하나를 고르는 문제는 **이진 분류(binary classification)** 
```python
import matplotlib.pyplot as plt   # 과학계산용 그래프 패키지
plt.scatter(bream_length, bream_weight)  # 산점도 그리는 함수
plt.xlabel('length') # X축은 길이
plt.ylabel('weight') # Y축은 무게
plt.show()
```
길이와 무게 두가지 특성<span class="sup-word">feature</span>으로 그린 그래프를 2차원 그래프라고 함
산점도 그래프가 일직선에 가까운 형태로 나타나는 경우를 **선형(linear)적** 이라고 함
