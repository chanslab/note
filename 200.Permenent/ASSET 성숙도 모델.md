## AI Strategy Stages and Evaluation Tool
NIA가 제시하는 AX성숙도 모델

- 1단계 : 인식(Awareness)
- 2단계: 기반 구축(Setup)
- 3단계: 활용 전면화(Systemization)
- 4단계: 고도화(Enhancement)
- 5단계: 전환 완료(Transformation)


### 1단계 → 2단계
리더의 역할이 결정적
리더가 직접 선언하고, 자원을 배분하고, 책임자를 지정하는 등 <mark style="background: #BBFABBA6;">조직을 재정비</mark>

### 2단계 → 3단계
프로세스 전면 재설계 필요
AI를 전제로 업무를 다시 설계 ≠ 기존 업무를 그대로 두고 AI를 얹는게 아님

### 3단계 → 4단계
가이드라인과 품질 관리가 핵심
AI활용이 확대되면 혼란도 따름
이때 체계적인 거버넌스가 필요

### 4단계 → 5단계
성과관리와 지속적인 혁신이 필요
AI가 실제로 성과를 내고 있는지 측정하고 계속 개선해야 함
