## Python 리스트(List)의 특징
### 1. 순서가 있는 자료구조 (Ordered)
리스트는 요소들이 **삽입된 순서를 유지**
```python
my_list = [1, 2, 3]
print(my_list[0])  # 1 (인덱스로 접근 가능)
```
### 2. 변경 가능 (Mutable)
생성 후에도 요소를 **추가, 수정, 삭제** 할 수 있음
```python
my_list = [1, 2, 3]
my_list[0] = 10       # 수정
my_list.append(4)     # 추가
my_list.remove(2)     # 삭제
```
### 3. 중복 허용 (Duplicates Allowed)
동일한 값을 **여러 번 저장**할 수 있음
```python
my_list = [1, 1, 2, 2, 3]  # 중복 가능
```
### 4. 다양한 자료형 혼합 가능 (Heterogeneous)
한 리스트 안에 **여러 타입**을 함께 담을 수 있음
```python
mixed = [1, "hello", 3.14, True, [1, 2]]
```
### 5. 동적 크기 (Dynamic Size)
크기가 **고정되어 있지 않아** 자유롭게 늘리거나 줄일 수 있음
```python
my_list = []
my_list.append(1)  # 필요할 때 요소 추가
```
### 6. 인덱싱 & 슬라이싱 지원
음수 인덱스와 슬라이싱으로 **유연하게 접근** 가능
```python
my_list = [10, 20, 30, 40, 50]
print(my_list[-1])    # 50 (마지막 요소)
print(my_list[1:3])   # [20, 30] (슬라이싱)
```
### 7. 내장 메서드 풍부
자주 쓰는 주요 메서드들:

| 메서드            | 설명           |
| -------------- | ------------ |
| `append(x)`    | 끝에 요소 추가     |
| `insert(i, x)` | 특정 위치에 삽입    |
| `remove(x)`    | 특정 값 삭제      |
| `pop(i)`       | 특정 위치 요소 꺼내기 |
| `sort()`       | 정렬           |
| `reverse()`    | 역순 정렬        |
| `len()`        | 길이 반환        |
