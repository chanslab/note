## 원-핫 인코딩

- 원시 텍스트(raw text)
```mermaid

graph TD
    subgraph Data [데이터 수집]
        A[원시 텍스트 Raw Text] --> B[말뭉치 Corpus]
    end

    subgraph Sparse [통계 기반 희소 표현]
        B --> C["원-핫 인코딩 (One-hot Encoding)"]
        C --> D["단어 가방 (Bag of Words, BOW)"]
        D --> E["TF-IDF"]
    end

    subgraph Dense [신경망 기반 밀집 표현]
        E --> F["워드투벡 (Word2Vec)"]
        F --> G["의미적 밀집 벡터 (Dense Vector)"]
    end

    subgraph Service [실무 시스템 배치]
        G --> H["벡터DB (Vector Database)"]
        H --> I["RAG / LLM 검색"]
        I --> J["API 서빙 (Serving)"]
    end

    %% 은은한 파스텔 톤 색상 정의 (Muted Pastel Tones)
    style Data fill:#F1F3F5,stroke:#DEE2E6,stroke-width:1px
    style Sparse fill:#EBF8FF,stroke:#BEE3F8,stroke-width:1px
    style Dense fill:#F0FFF4,stroke:#C6F6D5,stroke-width:1px
    style Service fill:#FFF5F5,stroke:#FED7D7,stroke-width:1px
    
    style A fill:#FFFFFF,stroke:#ADB5BD
    style F fill:#E6FFFA,stroke:#B2F5EA,stroke-width:2px,font-weight:bold

```
